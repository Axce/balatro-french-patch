# Balatro French Patch

Une modeste version communautaire de la traduction française de Balatro.

[![Balatro](https://www.playbalatro.com/press-kit/Logos/Balatro%20logo.png)](https://www.playbalatro.com/)

## Fork de https://github.com/FrBmt-BIGetNouf/balatro-french-translations

### Différences

- Corrections
- Porté vers [Lovely](https://github.com/ethangreen-dev/lovely-injector)

### Simplifications / régressions :

- Aucune image n’est traduite en français.

## Installation

1. Installer [Lovely](https://github.com/ethangreen-dev/lovely-injector)
2. Télécharger [le patch](https://github.com/Axce/balatro-french-patch/archive/refs/heads/main.zip)
3. Placer le dossier `balatro-french-patch` contenu dans l’archive dans `%APPDATA%/Balatro/Mods`
